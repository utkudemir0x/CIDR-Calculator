# 📖 About

CIDR Calculator is a lightweight command-line networking tool written in Python.

It allows you to quickly calculate IPv4 subnet information, inspect IP addresses, convert IP addresses to binary, and perform common networking calculations without requiring any external libraries.

- 🌐 CIDR Calculator

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/utkudemir0x/CIDR-Calculator.git
```

Enter the directory

```bash
cd CIDR-Calculator
```

Run

```bash
python cidr_calculator.py
```

or

```bash
python3 cidr_calculator.py
```

# 💻 Example

Input

IP Address:
